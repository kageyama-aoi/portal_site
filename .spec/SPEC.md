# SPEC - 技術仕様・要件定義

## 機能要件

### 1. ワークフロービュー：ステップのメモをホバーで表示
- ワークフロービューのステップカードにリンクの `memo` フィールドをツールチップ表示
- 対象：ワークフロービューのみ（カード・テーブルビューは対象外）
- 実装：CSS `title` 属性 または カスタムツールチップ

### 2. Explorer リンク / URL リンクの色による視覚的切り分け
- `opendir:` リンク → 緑系（現行の `--color-local` ティール）をより明確に
- 通常 URL リンク → 青系（現行の `--primary` ブルー）をアクセントに追加
- 現在は `opendir:` のみ左ボーダーあり。通常 URL にも左ボーダーを追加して並列で区別
- アイコンの変更は行わない

### 3. ヘッダーの表示切替ボタン並び順を変更可能にする
- 現在の固定順：カード / テーブル / 思い出し / 作業フロー
- ユーザーが並び順をドラッグ＆ドロップで入れ替えできるようにする
- 並び順は `localStorage` に保存し、次回起動時も維持

## 非機能要件
- 既存のデータモデル（data.json）変更なし
- vanilla JS のみ、フレームワーク導入なし

## 技術構成
- 対象ファイル：`js/ui.js`, `js/dialogs/workflowDialog.js`, `style.css`
- ボタン並び順の保存キー：`portalViewBtnOrder`（localStorage）
